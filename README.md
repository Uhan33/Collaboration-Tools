### ✨프로젝트 소개

### 👨‍👩‍👦‍👦 멤버구성 & 맡은 기능
- 최인철: 회원가입 / 로그인 / 조회 / 수정 / 탈퇴
- 이재헌: 보드 생성 / 조회 / 수정 / 삭제  + 생성된 보드에 팀원초대 + 초대 승낙 및 거절
- 조완희: 리스트 생성 / 조회 / 수정 / 삭제 + 리스트 순서 변경
- 정유한: 카드 생성 / 조회 / 수정 / 삭제 + 카드 순서 변경 + 파일 업로드
- 정태성: 댓글 생성 / 조회 /수정 / 삭제
